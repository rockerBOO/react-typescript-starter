# React Starter

React 18 starter
- Typescript 4
- Yarn 3 (node_modules linker)
- Vite 3
- ESLint 8,
- Stylelint 14,
- Jest 29
- Testing Library React 13
- Prettier 2

This is what I currently like to use as a starter mix to make a web application. Sets up a page with react and vite.

## Install

``` 
yarn install
```

## Usage

```
yarn dev
```

```
yarn build
```

```
yarn test
```
